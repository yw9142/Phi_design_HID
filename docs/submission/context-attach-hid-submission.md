# Context Attach HID Submission

## HID 프로토타입 이름

Context Attach — 필요한 과거 대화만 이어 붙이는 LLM 협업 공간

## 사용자, 페인포인트, 목표 설명

여러 프로젝트를 동시에 진행하는 PM·기획자는 중요한 결정 근거가 여러 LLM 대화에 흩어진 채 다음 문서를 만들어야 합니다. 새 질문을 시작할 때마다 과거 채팅을 검색하고, 필요한 부분을 읽어 다시 설명하느라 사고 흐름이 끊깁니다. 맥락을 빠뜨리면 답변 품질이 떨어지고, 자동 기억에 맡기면 어떤 정보가 사용됐는지 확인하기 어렵습니다. 사용자의 목표는 지금 질문에 필요한 과거 대화만 추천받아 선택적으로 붙이고, 근거와 연결 관계를 확인하며 바로 다음 의사결정으로 이어가는 것입니다.

## 핵심 마찰 설명

현재 질문에 필요한 과거 맥락을 사용자가 기억해 직접 찾고 복사해야 하며, 이 회수 비용이 LLM과의 작업 연속성을 끊습니다. 반대로 맥락을 자동으로 섞으면 어떤 정보가 답변에 쓰였는지 통제하고 검증하기 어렵습니다.

## 키스크린 한 장

https://context-attach-hid.vercel.app/

## HID 프로토타입 작동 환경

- 모바일
- 데스크탑

## 데모 흐름

1. 현재 질문을 바탕으로 관련 과거 대화와 추천 이유를 확인합니다.
2. 답변에 사용할 맥락만 명시적으로 첨부합니다.
3. 첨부한 근거로 후속 질문을 보내고, 구조화된 답변과 출처를 확인합니다.
4. 오른쪽 그래프에서 관계를 탐색하고 노드를 눌러 원문 대화로 이동합니다.
